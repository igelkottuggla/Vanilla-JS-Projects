## CountDown project

Simple count down that shows time remaining before the event ends

Used:

-   if...else statements
-   Date() constructor
-   getUTCFullYear(), getMonth(), getDate(), getHours(), getFullYear(), getMinutes(), getDay() methods
-   Template literals
-   Math.floor() function
-   remainder operator
-   forEach() method
-   innerHTML property
-   setInterval() method
-   arrow functions
